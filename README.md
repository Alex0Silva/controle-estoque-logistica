![Objetivo](https://img.shields.io/badge/OBJETIVO-0A1AFF?style=for-the-badge)

Criar um sistema simples de controle de estoque que represente situações reais do dia a dia, como:
- Cadastro de produtos  
- Controle de quantidade  
- Entrada e saída de itens  
- Identificação de produtos vencidos  

O foco é praticar lógica de programação e transformar um problema real em uma solução digital.
